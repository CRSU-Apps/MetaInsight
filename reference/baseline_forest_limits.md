# Extract the default x-axis limits using the same method internal to `gemtc::forest()`

Extract the default x-axis limits using the same method internal to
[`gemtc::forest()`](https://rdrr.io/pkg/gemtc/man/blobbogram.html)

## Usage

``` r
baseline_forest_limits(forest_data)
```

## Arguments

- forest_data:

  data.frame. Created by
  [`baseline_forest_format()`](https://apps.crsu.org.uk/MetaInsight/docs/reference/baseline_forest_format.md)
