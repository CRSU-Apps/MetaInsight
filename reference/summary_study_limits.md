# Find the default minimum and maximum values for the x-axis and calculate a sensible step to use in the numeric input

Find the default minimum and maximum values for the x-axis and calculate
a sensible step to use in the numeric input

## Usage

``` r
summary_study_limits(pairwise, outcome)
```

## Arguments

- pairwise:

  Results of pairwise analysis

- outcome:

  character. Outcome type for the dataset. Either `binary` or
  `continuous`.

## Value

Vector of xmin and xmax
